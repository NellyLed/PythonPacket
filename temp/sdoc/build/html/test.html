CHAPTER 1 Вычисления¶

Вывод сообщения¶

def dialogue(name): ‘’’ Выводит на экран выражение “I purple you” с добавлением введенного имени(форма приветствия). Параметры: name (string): имя пользователя Возвращаемое значение: name_str = I purple you,name name_str (str): строка,выводящая предложение на экран ‘’’

Расчет числа Фиббоначи¶

def fibon(n): ‘’’ Выводит ряд чисел, в котором каждое последующее число равно сумме двух предыдущих чисел. Параметры: n(int): максимальное значение, на котором заканчивается счет Возвращаемое значение: enter = [x for x in fibon(15)] enter (str): ряд чисел (объект-генератор) ‘’’

Сумма двух чисел¶

def add(a, b, c, d): ‘’’ Возвращает сумму двух целых чисел. Параметры: a(int): первое целое число b(int): второе целое число c(int): третье целое число d(int): четвертое целое число Возвращаемое значение: add_sum = a + b + c + d add_sum(str): строка суммы a,b,c и d ‘’’

Произведение двух чисел¶

def mul(k, w, q, l) ‘’’ Вычисляет произведение двух целых чисел и выводит результат на экран. Параметры: k(int): первое целое число w(int): второе целое число q(int): третье целое число l(int): четвертое целое число Возвращаемое значение: mul_s = k * w * q * l mul_s(str): строка произведения k,w,q и l ‘’’

Вычитание двух чисел¶

def sub(e,f,g): ‘’’ Вычисляет результат вычитания двух целых чисел и выводит результат на экран. Параметры: e(int): первое целое число f(int): второе целое число g(int): третье целое число Возвращаемое значение: sub_s = e - f - g sub_s(str): строка вычитания e,f и g ‘’’

Деление двух чисел¶

def div(p,m): ‘’’ Вычисляет результат деления двух целых чисел и выводит результат на экран. Параметры: p(int): первое целое число m(int): второе целое число Возвращаемое значение: div_s = p / m div_s(str): строка деления числа p на m ‘’’
